零件及零件图

零件设计构型分类

根据零件的功能，其结构可以分为以下几个方面：

  1. 支承结构：轴承孔、轴孔、支承面等；
  2. 连接结构：连接螺栓通孔、螺孔、键槽、轮毂等；
  3. 定位结构：销孔，轴肩，定位面等；
  4. 传动结构：各类轮齿、皮带轮等；
  5. 加强结构：肋板等；
  6. 润滑结构：储油池，注油孔，排油孔，油标孔，集油槽等；
  7. 密封结构：密封面，密封槽等；
  8. 工艺结构：退刀槽，越程槽，凸台，凹坑，圆角，倒角，轴端螺孔，中心孔等

螺纹

  1. 要素
  1. 一个与轴线共面的平面图形（三角形、梯形等），绕圆柱面作螺旋运动，则得到一圆柱螺旋体;
  2. 外表面叫做外螺纹，内表面是内螺纹
  3. 螺距与导程：螺距是最近相邻两个螺纹之间的距离；导程是

顶径： 指的是外螺纹的大径或者内螺纹的小径；

螺纹顶径，是指与外螺纹或内螺纹牙顶相切的假想圆柱或圆锥的直径。

中径：一个假想圆柱的直径。该圆柱的母线通过牙型上沟槽和凸起宽度相等的地方。

公称直径：指的是螺纹大径的基本尺寸

简单理解(重点)

大径小径即顾名思义，大的叫大径，小的叫小径；

公称直径 = 大径；

顶径指的是露在外面的螺纹所对应的直径就叫顶径。

齿轮基本参数

齿顶圆\(d_a\)

齿根圆\(d_f\)

分度圆\(d\)

齿距\(p\)

齿数\(z\)：一个齿轮的轮齿总数

模数\(m\)：齿轮的齿数z、齿距p和分度圆直径d之间有如下关系：

\(\pi d = zp 即 d = zp/ \pi\)

​ 由于式中出现了无理数\(\pi\)，不便计算和标准化，\(令p/ \pi= m，则d = mz\)

模数\(m\)：是设计、制造齿轮的重要参数。m值越大表示齿轮承载能力越大。 为了便于设计加工，模数的值已标准化

齿高：(齿顶高系数)\(h_a\)：齿轮的齿顶高通常规定为与模数成一定的比，该比值称为齿顶高系数， ha = ha /m，我国规定正常齿轮的齿顶高系数\(h_a\) = 1，因此有：$ h_a = h_a m = m$。

齿根高：\(h_f = 1.25m\)

齿顶角\(\alpha\)：齿廓曲线在分度圆上的一点处的速度方向与曲线在该点处的法线方向之间所夹锐角称为齿形角。齿形角也已经标准化，我国规定\(\alpha = 20º\)，即规定分度圆为通过齿廓曲线上齿形角为\(20º\)的点所作的圆。

传动比：\(i\)：\(i = n_1 / n_2 = z_2 / z_1\)

零件图

表面结构要求

表面粗糙度:

定义：

在加工零件时，由于刀具在零件表面上留下的刀痕及切削分裂时表面金属的塑性变形所形成的具有较小间距（通常波距小于1mm）的峰谷所组成的微观几何形状特性。

表面波纹度
形状误差

表面结构是原始轮廓（P轮廓）、粗糙度轮廓（R轮廓）和波纹度轮廓（W轮廓）的叠加

评定表面结构的轮廓参数

  1. 轮廓参数是工程图样中常用的评定表面结构的参数，它包含了粗糙度轮廓参数（R参数）、波纹度轮廓参数（W参数）和原始轮廓参数（P参数）。

  2. 评定粗糙度轮廓参数: Ra和Rz，其中Ra是最常用的评定参数。

  3. 轮廓算术平均偏差——Ra：在取样长度l范围内，被测轮廓上各点至基准线的距离yi的算术平均值。

  5. 表面结构符号

    细捋

    1. 第一个是任何方法获得
    2. 第二个是去除材料的方法获得
    3. 第三个是不用去除材料的方法得到
    4. 第四个是加横线来标注参数和说明
    5. 第五个加圆圈是表示所有表面具有相同表面粗糙程度要求
    6. 对于标注要求（就是除了参数以外的那个东西），有两个：
      1. 不标注的话默认16%规则：实测不超过总数16%个不合格的
      2. 标注max则是最大值规则：实测不能超过给定的上限值，即不超过总数0%个不合格的
    7. 注意Ra的单位是\(\pmb{\mu m}\)

极限与配合

偏差
  • 公称尺寸
  • 极限尺寸
  • 实际尺寸

尺寸偏差：

  1. 上极限偏差 = 上极限尺寸-公称尺寸，孔用ES，轴用es表示
  2. 下极限偏差 = 下极限尺寸-公称尺寸，孔用EI，轴用ei表示
  3. 公差：上-下
公差带
标准公差和基本偏差

  1. 国家标准《极限与配合》规定了公差带由标准公差和基本偏差两个要素组成。标准公差确定公差带的大小，而基本偏差确定公差带的位置。

  2. 标准公差的数值由公称尺寸和公差等级确定，公差等级分20个等级：IT01、IT0、IT1～IT18，精度依次降低。

  3. 基本偏差确定公差带相对于零线的位置，一般为靠近零线的那个极限偏差，代号: 孔用大写字母，轴用小字母表示，各28个。

配合

  1. 公称尺寸相同相互结合的孔和轴的公差带之间的关系。

    \(δ=孔的实际尺寸－轴的实际尺寸,δ≥0 间隙 \ \ \ \ δ≤ 0 过盈\)

  2. 种类：

    1. 过盈配合
    2. 过渡配合
    3. 间隙配合

  3. 配合制

    • 基孔制配合 
      •  基本偏差为一定的孔的公差带，与不同基本偏差的轴的公差带形成各种配合的制度。基孔制的孔称为基准孔，其基本偏差代号为Ｈ，其**下偏差**为零。
    • 基轴制配合 
      • 基本偏差为一定的轴的公差带与不同基本偏差的孔的公差带形成各种配合的制度。基轴制的轴称为基准轴，其基本偏差代号为ｈ，其**上偏差**为零。

2. 工图琐碎知识点

极限与配合

  1. 极限偏差注意大小写

  1. 公差带

    标准公差确定公差带的大小，而基本偏差确定公差带的位置。

  2. 圆角：防止冷却时产生缩孔或裂纹；防止脱模时砂型落砂

  3. 钻孔端面：避免钻孔偏斜和钻头折断

  4. 凸台、凹坑：减少机械加工量/保证表面接触

  5. 球尺寸标注：\(SR,\ \ S\phi\)

  7. 螺纹尺寸标记：5g6g表示中+顶径（普通螺纹），其他螺纹只表示中经

  8. 特征代号公称直径×导程（P螺距）旋向-公差带代号-长度代号

  9. 1：2是画大还是画小——画图：实际，所以是变小了

  11. 工艺结构：圆角、倒角、退刀槽、砂轮越程槽、钻孔端面、凸台凹坑

  14. 区分弹簧垫圈（从左上到右下60度，小于螺母）和普通垫圈（大于螺母）
