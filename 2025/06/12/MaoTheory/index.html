¶第一章

毛泽东思想是马中华的第一果，是关于搞革命、搞改造、搞建设的理论

¶毛泽东思想的形成和发展的历史条件

四个条件，记忆方法：

  • 马中化对应指导思想
  • 搞革命搞改造搞建设——说明我国革命和建设的特殊国情+实践 （因为马中化） 经验
  • 与此同时世界各国都在革命——世界进入帝国主义和无产阶级革命时代
  1. 时代背景：世界进入帝国主义和无产阶级革命时代（战争与革命的时代）
  2. 指导思想：十月革命给中国送来了马克思列宁主义（中国革命科学的指导思想）
  3. 特殊国情：中国革命和建设的特殊国情
  4. 实践基础：中国革命和建设的实践经验

¶毛泽东思想形成和发展的过程

三阶段

  • 初步形成
    1. 大革命时期：提出新民主主义革命的基本思想
    2. 土地革命时期：开辟了农村包围城市、武装夺取政权的革命道路
  • 成熟时期：
    1. 走向成熟：遵义会议到抗战时期
    2. 指导思想：1945年党的七大，毛思想被确立为党的指导思想
  • 丰富和发展：解放战争时期和中华人民共和国成立后

¶毛泽东思想的科学内涵

  1. 毛泽东思想是马克思主义中国化第一次历史性飞跃的理论成果，是马列主义在中国的运用和发展

马中化的第一果👆 第二果是中国特色社会主义体系概论 最新成果是习近平新时代中国特色社会主义

  1. 是被实践证明了的关于中国革命和建设的正确的理论原则和经验总结**(是正确的)**
  2. 是中国共产党集体智慧的结晶，毛泽东是这个理论体系的主要创造者**（是集体的）**

¶毛泽东思想的主要内容

  1. 新民主主义革命理论——是毛思想达到成熟的主要标志

    是革命的指导思想👆

  2. 党的建设理论——特别强调要把思想建设放在首位

    中共的三大优良作风——

    1. 理论联系实际
    2. 密切联系群众
    3. 批评与自我批评的作风

  3. 社会主义革命和社会主义建设理论

  4. 人民军队建设和军事战略理论

  5. 政策和策略理论

  6. 思想政治工作和文化工作理论

¶毛思想活的灵魂

毛思想活的灵魂：实事求是、群众路线、独立自主

  1. 实事求是（一切从实际出发，理论联系实际，坚持在实践中检验真理和发展真理）

    • 地位：党的思想路线
      • 马中化成果的精髓和灵魂
      • 毛思想的精髓和灵魂
      • 中国特色社会主义理论体系的精髓和灵魂

  2. 群众路线：一切为了群众、一切依靠群众、从群众中来、到群众中去

    • 地位：党的根本工作路线

  3. 独立自主：（依靠自己的能力）从中国实际触发，依靠本国革命力量和恶人民群众的努力，走出适合本国特点的革命与建设道路

¶毛思想的历史地位

  • 毛泽东第一个提出了“马克思主义中国化”的科学命题和重大任务
  1. 毛思想是马中化的第一个重大理论成果
  2. 毛思想是中国革命和建设的科学指南
  3. 毛思想是中华民族宝贵的精神财富
  4. 毛思想为中国特色社会主义理论体系的形成奠定了思想理论基础

二、毛概划重点

¶cxw老师

¶导论部分

选择题可能的考点：

  1. 六届六中全会——第一次明确提出马克思主义中国化的要求

¶一、为什么推进马克思主义中国化、时代化？

  1. 马克思主义中国化、本土化==才能深入人心==
  2. 马克思主义中国化同时包含着马克思主义时代化的意蕴
  3. 推进马克思主义中国化、时代化，是马克思主义理论本身发展的==内在要求==
  4. 推进马克思主义中国话、时代话，是==解决中国实际问题的客观需要==

¶二、马克思主义中国化时代化的内涵

  1. 运用马克思的立场、观点和方法，观察时代、把握时代、引领时代，解决中国革命、建设、改革的实际问题
  2. 总结和提炼中国革命、建设、改革的实践经验并将其上升为理论，不断丰富和发展马克思主义的理论宝库，赋予马克思主义以新的时代内涵
  3. 运用中共人民喜闻乐见的民族语言来阐述马克思主义，使其根植于中华优秀传统文化的土壤之中，具有中国特色，中国风格，中国气派

¶三、坚持和发展马克思主义的两个结合==（重要）==

  • 坚持和发展马克思主义，必须同中国具体实际相结合、同中华优秀传统文化相结合

    革命、建设、改革是个关键词

    解放思想、实事求是、与时俱进、求真务实

  1. 要坚持解放思想、实事求是、与时俱进、求真务实，一切从实际出发，着眼解决革命、建设、改革中的实际问题，作出符合中国实际和时代要求的正确回答，得出符合客观规律的科学知识，形成与时俱进的理论成果，更好的指导中国实践
  2. 要坚定历史自信、文化自信，坚持古为今用、推陈出新，把马克思主义思想精髓同中华优秀传统文化精华贯通起来、同人民群众的共同价值观念融通起来。

¶四、马克斯主义中国化历史进程——一脉相承、与时俱进

一脉相承、与时俱进是关键词，马克思主义中国化、时代化的关系

主要分为四个阶段：

  1. 新民主主义革命时期——成果是新民主主义革命理论
  2. 社会主义革命和建设时期——成果是社会主义改造理论
  3. 改革开放和社会主义现代化建设新时期——成果是以邓小平理论、三个代表重要思想、科学发展观为代表的中国特色社会主义理论体系的确立
  4. 中国特色社会主义新时代——成果是习近平新时代中国特色社会主义思想

¶第一章 ⽑泽东思想及其历史地位

¶一、毛泽东思想的主要内容

六个部分——最多客观题考一下（eg:哪些内容不属于毛泽东思想的主要内容）

但是建议掌握，是主要框架

  1. 新民主主义革命理论
  2. 社会主义革命和社会主义建设理论
  3. 革命军队建设和军事战略的理论
  4. 政策和策略的理论
  5. 思想政治工作和文化工作的理论
  6. 党的建设理论

¶二、毛泽东思想活的灵魂

三个方面，重要 （三也是一个很重要的数字）

党的作风建设三个要求：理论联系实际、密切联系群众、开展批评和自我批评

毛泽东思想新民主主义革命的三大法宝：统一战线、武装斗争、党的建设

  1. 实事求是：就是⼀切从实际出发，理论联系实际，坚持在实践中检验真理和发展真理
  2. **群众路线：**就是一切为了群众，一切依靠群众，从群众中来，到群众中去，把党的正确主张变为群众的自觉行动。
  3. **独立自主：**就是坚持独立思考，走自己的路，就是坚定不移地维护民族独立、捍卫国家主权，把立足点放在依靠自己力量的基础上，同时积极争取外援，开展国际经济文化交流，学习外国一切对我们有益的先进事物

¶毛泽东思想的历史地位

分点论述的框架：理论的、历史的、现实的

  1. **理论价值：**马克思主义中国化时代化的第一个重大理论成果
  2. **历史价值：**中国共产党和中国人民宝贵的精神财富
  3. **现实价值：**中国革命和建设的科学指南

¶新民主主义革命理论

¶一、近代中国的基本国情

主要内容：半殖民地半封建社会，以下为依据（考察概率不大）：

  • **半封建国情依据：**帝国主义的侵略虽然在⼀定程度上促使封建社会⾃给⾃⾜的⾃然经济解体，在客 观上为中国资本主义的发展创造了⼀定条件，但中国不可能发展成为资本主义国家。
  • **半殖⺠地国情依据：**帝国主义列强通过政治的、经济的和⽂化的侵略，使中国半殖⺠地化。
  • **主要⽭盾：**帝国主义和中华⺠族的⽭盾、封建主义和⼈⺠⼤众的⽭盾。

¶二、新民主主义革命的总路线

非常重要

总路线：⽆产阶级领导的，⼈⺠⼤众的，反对帝国主义、封建主义和官僚资本主义的⾰命

不同历史阶段⾰命的主要对象和统⼀战线：

  1. 国共合作的⼤⾰命时期：帝国主义⽀持下的北洋军阀 (国共合作统⼀战线)
  2. ⼟地⾰命战争时期： 国⺠党新军阀 (军⺠合作统⼀战线)
  3. 抗⽇战争时期： ⽇本帝国主义 (抗⽇⺠族统⼀战线)
  4. 解放战争时期： 美帝国主义支持下的国民党反动派 (人民民主统一战线)

新⺠主主义⾰命的动⼒: 无产阶级、农民阶级、城市小资产阶级和民族资产阶级

  1. 无产阶级是中国革命最基本的动力
  2. 农民是中国革命的主力军
  3. 城市小资产阶级也是无产阶级的可靠同盟者
  4. ⺠族资产阶级也是中国⾰命的动⼒之⼀。(两⾯性)

区分新旧⺠主主义⾰命：均有资产阶级参加，重点为新⺠主主义⾰命中⽆产阶级掌握⾰命的领导权

⭐新⺠主主义⾰命的性质：

  1. 性质： 是资产阶级民主主义革命
  2. 新民主主义革命和社会主义革命性质不同
  3. 新民主主义革命仍然属于资产阶级民主主义革命的范畴。他要建立的是无产阶级领导各革命阶级的联合专政，而不是无产阶级专政。
  4. 社会主义革命是无产阶级性质的革命，他所要实现的目标是消灭资本主义剥削制度和改造小生产的私有制。
